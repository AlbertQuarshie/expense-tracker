# Expense-Tracker

This project is the creation of a financial display website designed to help users track daily spending.

## Contributor

Albert Junior Quarshie

### A. Project Brief

The Expense Tracker is a functional web application that allows users to log expenses, categorize their spending, and visualize their financial behaviour. It utilizes local persistence to ensure user data remains available across browser sessions.

### B. Core Features

1. Smart Logging
   Users can add expenses with descriptions, amounts, and categories. The system automatically assigns unique IDs to every entry for precise data management.

2. Categorized Values
   A dynamic filtering system allows users to view specific spending habits (e.g., Food, Transport, Entertainment) or their total daily expenditure at a glance.

3. Persistent Memory
   By integrating localStorage, the application ensures that your financial history isn't lost when you refresh the page or close your browser.

4. Automated Calculations
   Using functional programming, the app instantly calculates totals and updates the UI in real-time as items are added or removed.

### C. Technologies Used

HTML: For semantic document structure.

CSS: For a responsive, modern, and user-friendly interface.

JavaScript: Implementing OOP, Destructuring, and Spread/Rest operators.

Jest: For rigorous unit testing of the core logic.
